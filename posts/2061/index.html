VS Code 常用快捷键

发表于 2017-08-26 更新于 2019-03-27 分类于 收集控

前言

VS Code 还是我用的比较顺手的一个编辑器了，此篇文章记录下我常用的以及以后可能用到的快捷键。

通用

KeyValue
Ctrl+Shift+P, F1显示所有命令
Ctrl+P快速打开
Ctrl+Shift+N新建窗口
Ctrl+Shift+W关闭窗口

编辑

KeyValue
Alt+ ↑ / ↓移动行
Shift+Alt + ↓ / ↑复制行
Ctrl+Shift+K删除行
Ctrl+Shift+\跳转到匹配的括号
Ctrl+] / [缩进/取消缩进行（比TAB好用）
Ctrl+Home/End跳转到文件开头/结尾
Ctrl+↑ / ↓滚动
Ctrl+Shift+[折叠/显示代码块
Ctrl+/行注释
Shift+Alt+A块注释

导航

KeyValue
F8 / Shift+F8跳转到下一个/上一个错误或警告处
Ctrl+F/H搜索/替换
F3 / Shift+F3下一个/上一个结果

多光标和区块

KeyValue
Alt+Click插入光标
Ctrl+U上一次的光标操作
Shift+Alt+I在选择的每一行的后面插入光标
Ctrl+I选择当前行

富文本编辑

KeyValue
Ctrl+Space触发建议
Ctrl+Shift+Space触发参数提示

编辑器

KeyValue
Ctrl+\分割编辑器

文件管理

KeyValue
Ctrl+K P复制当前文件的路径
Ctrl+K R在资源管理器中显示当前文件

调试

KeyValue
F9断点
F5开始/继续
Shift+F5停止
F11 / Shift+F11Step into/out
F10Step over

内置终端

KeyValue
Ctrl+`显示内置终端
Ctrl+Shift+`创建新的终端
Ctrl+Shift+C复制选中部分
Ctrl+Shift+V粘贴
