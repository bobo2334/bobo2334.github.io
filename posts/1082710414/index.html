Windows 10 开机自动校时

发表于 2019-02-26 更新于 2019-03-27 分类于 代码宅

前言

现在在一台 PC 上同时装了 Windows 10 和 MacOS，遇到了一个关于系统时间的问题。

当从 MacOS 重启到 Windows 系统时，Windows 系统的时间会变得不正确，我也不知道是什么原因导致了这个问题。每次我都需要手动点一下同步时间，但是 Windows 系统自动同步时间的周期太长，每次手动更新时间也挺麻烦，所以我就在思考有没有什么办法能在开机的时候自动校时。

开始的时候尝试了网络上的一些校时软件，但是我想到了一种更为优雅的方式，这里记录一下通过Windows 计划任务的方式在开机的时候自动同步网络时间。

启动 Windows Time 服务

运行services.msc命令或者在小娜处搜索服务来打开服务管理窗口。

在服务列表里找到Windows Time，开启此服务并设置为开机启动。

修改 NTP 服务器

NTP 服务器提供标准时间，Windows 默认的服务器很多时候都同步失败，为了解决这个恼人的问题我们可以换一个稳定点的 NTP 服务器。

在右键点击时间->调整日期/时间->日期、时间和区域格式设置->Internet 时间->更改设置中，勾选与 Internet 时间服务器同步，并在服务器后的输入框中填入 NTP 服务器地址。

国内稳定的 NTP 服务器推荐（道听途说）：

  • pool.ntp.org
  • 0.cn.pool.ntp.org
  • ntp1.aliyun.com

在小娜处搜索计划任务来打开计划任务管理窗口。
