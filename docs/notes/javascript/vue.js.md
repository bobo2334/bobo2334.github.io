## Vue 简介

Vue.js 是一套**构建用户界面**的**渐进式** JavaScript 框架。

- **构建用户界面**：将数据渲染在页面上；
- **渐进式**：Vue 可以自底向上逐层应用。如果是简单应用只需要引入核心库；如果是复杂应用，可以引入各种各样的 Vue 插件。
