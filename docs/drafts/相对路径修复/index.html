相对路径修复

Typora 插入图片时使用的是/xxx.assets/xxx.png格式的路径，但是 Vuepress 需要用./xxx.assets/xxx.png这种格式的路径，不然会被解析到网站的根目录。

使用 VS Code 的正则表达式替换解决。

搜索\]\((?!\.|http)，替换为](./。
