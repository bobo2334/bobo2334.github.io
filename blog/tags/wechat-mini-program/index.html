使用 Vant Weapp 自定义微信小程序 TabBar

2020年12月8日

前言​

微信小程序原生的 TabBar 虽然设置简单，但是样式不好看，也不能使用字体图标，但是微信小程序有自定义 TabBar 的机制。

Vant WeApp1 是有赞出品的针对微信小程序的一套开源组件库，结合 van-tabbar2 和微信的自定义 TabBar 机制 3 可以实现一套比较好看的 TabBar。
